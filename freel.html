Custom Library Links

Custom Library Links allow you to create a block in your Canvas course with links to the library resources you think are most useful for your students. This wizard will walk you through selecting the links and adding the block to your class.

1. Select the links you want to appear.

3. Paste the code onto a Page in your Canvas course, using the "HTML Editor" view.

    Open your Canvas course, select the Page where you want the widget, and click the Edit button. Switch to the HTML Editor view, then simply paste the code. Once added, click the Save (or Save and Publish) button.
